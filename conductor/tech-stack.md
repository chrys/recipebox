# RecipeBox Tech Stack

## Backend
- **Language:** Python
- **Framework:** Django (>= 5.1)
- **Database:** PostgreSQL (via psycopg2-binary)
- **Environment Management:** python-dotenv

## Frontend
- **Templating:** Django Templates
- **UI Library:** Oat UI (Ultra-lightweight, zero dependency Vanilla HTML/CSS/JS)
- **Styling:** Vanilla CSS (integrated via Oat)
- **Interactivity:** Vanilla JavaScript

## Utilities & Media
- **Image Processing:** Pillow (for recipe image handling)
